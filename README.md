# CONTROLS
- Check your controls in the option menu to see `< control >` keybinds
  * Open Menu: `<Aim Down Sight>` && `<Melee Attack>`
  * Scrolling: Up \ Down: `<Aim Down Sight>` or `<Attack>`
  * Scrolling: Left \ Right: `<Lethal Offhand>` or `<Tactical Offhand>`
  * Select: `<Use>`
  * Back: `<Melee Attack>`
